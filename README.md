# Student Marks Application

This web application allows students to store their subject marks and calculate their results. It provides a user-friendly interface for entering marks, calculating totals and averages, and storing data for future reference.

## Project Structure

```
student-marks-app
├── src
│   ├── index.html          # Main HTML document
│   ├── css
│   │   └── styles.css      # Styles for the application
│   ├── js
│   │   ├── app.js          # Main JavaScript file
│   │   ├── calculator.js    # Functions for calculating results
│   │   └── storage.js      # Functions for storing and retrieving marks
│   └── assets
│       └── fonts           # Font files used in the application
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd student-marks-app
   ```

3. Install the necessary dependencies:
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the application.

## Usage Guidelines

- Enter your subject marks in the provided input fields.
- Click the "Calculate" button to see your total and average marks.
- Your marks will be stored in local storage, allowing you to retrieve them later.
